## useState and button click handler 

This was a simple project to get comfortable with `useState` and button click handlers.

```typescript

const [birthdays, setBirthdays] = useState(birthday_array)
    const clickHandler = () => {
        setBirthdays([]);
    }
    // ...
    // The button click handler:
    <button
        className="justify-center px-8 sm:px-16 md:px-24 rounded-md bg-pink-400 text-white py-2 focus:outline-none border border-transparent"
        type="button"
        onClick={clickHandler}>
        Clear all
    </button>
```

